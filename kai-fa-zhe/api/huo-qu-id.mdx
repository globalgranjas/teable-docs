本文介绍如如何在界面上获取 Teable 中的各类资源 Id

## SpaceId

点击目标空间，复制 URL 上以 spc 开头的字符串，示例: `spcXXXXXXXXXX`

## BaseId

点击目标数据库，复制 URL 上以 bse 开头的字符串，示例：`bseXXXXXXXXXX`

![](/images/kai-fa-zhe/api/files%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F3UAxYOALe4vISZRKICva%252Fuploads%252FL1B39vdGELFrgUv5ddOn%252Fimage.png)

## TableId

点击目标表格，复制 URL 上以 tbl 开头的字符串, 示例：`tblXXXXXXXXXX`

![](/images/kai-fa-zhe/api/files%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F3UAxYOALe4vISZRKICva%252Fuploads%252Fd9x1bIPU6C8wNIOC21DO%252Fimage.png)

## ViewId

点击目标视图，复制 URL 上以 viw 头的字符串, 示例：`viwXXXXXXXXXX`

![](/images/kai-fa-zhe/api/files%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F3UAxYOALe4vISZRKICva%252Fuploads%252FISm5UmFpxypxCtxfOH0u%252Fimage.png)

## FieldId

点击表格右上角齿轮按钮，进入设计界面，查看所有字段的 ID

![](/images/kai-fa-zhe/api/files%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F3UAxYOALe4vISZRKICva%252Fuploads%252FCFtHpivrWq8z2O0iDeeL%252Fimage.png)

## RecordId

展开记录编辑卡片，复制 URL 上 recordId= 后方以 rec 开头的字符串, 示例：`recXXXXXXXXXX`
