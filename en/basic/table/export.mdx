### Exporting Tables

Teable supports data export functionality, allowing users to download table data in their preferred format for easy data synchronization and backup.

#### Supported Export Formats

* CSV (Comma Separated Values)

#### How to Export Your Table

Click the icon next to the table and select "Download CSV". The downloaded file will be saved to your device's default download folder.
