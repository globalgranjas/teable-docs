## SpaceId

Click on the target space and copy the string starting with 'spc' from the URL, example: `spcXXXXXXXXXX`

![](/images/developer/api/fa9fb4d225e32565.png)

![](/images/developer/api/b7c3b29073bc6768.png)

## BaseId

Click on the target base and copy the string starting with 'bse' from the URL, example: `bseXXXXXXXXXX`

![](/images/developer/api/9f35f5ecaf38f264.png)

![](/images/developer/api/17f341d5b0b07ff4.png)

## TableId

Click on the target table and copy the string starting with 'tbl' from the URL, example: `tblXXXXXXXXXX`

## ViewId

Click on the target view and copy the string starting with 'viw' from the URL, example: `viwXXXXXXXXXX`

![](/images/developer/api/830177806519ba17.png)

## FieldId

Click the gear icon in the top right corner of the table to enter the design interface and view all field IDs

![](/images/developer/api/262744d8f8c9d0a3.png)

![](/images/developer/api/a2c6a5f136767885.png)

## RecordId

Expand the record edit card and copy the string starting with 'rec' after 'recordId=' in the URL, example: `recXXXXXXXXXX`

![](/images/developer/api/a7813345bcd4a8e3.png)

![](/images/developer/api/3a3abbbb21e81c7c.png)
